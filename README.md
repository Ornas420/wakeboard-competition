# Wakeboard Competition Management System

Monorepo for managing wakeboard competitions — registration, heat generation, scoring, and live results.

## Structure

- `/client` — React + Vite + TailwindCSS frontend
- `/server` — Node.js + Express + SQLite backend

## Getting Started

```bash
npm install
npm run dev
```

This starts both client (port 5173) and server (port 3000) concurrently.
